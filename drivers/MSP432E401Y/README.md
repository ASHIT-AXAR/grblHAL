## MSP432E401Y

A GrblHAL driver for the Texas Instruments [MSP432E401Y LaunchPad](http://www.ti.com/tool/MSP-EXP432E401Y#).

Default pin assignments matches the [CNC BoosterPack](https://github.com/terjeio/CNC_Boosterpack).
